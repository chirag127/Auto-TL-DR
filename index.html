How It Works

Auto TL;DR uses Google's Gemini 2.0 Flash Lite AI to generate high-quality summaries of web content. As you browse, the extension detects large blocks of text and offers to summarize them, saving you time and helping you get the key points quickly.

Installation

1

Clone the Repository

Get the code from GitHub:

git clone https://github.com/chirag127/Auto-TL-DR.git
2

Set Up the Backend

Navigate to the backend directory and install dependencies:

cd Auto-TL-DR/backend
npm install

Create a .env file with your Gemini API key:

GEMINI_API_KEY=your_api_key_here
PORT=3000

Start the server:

npm start
3

Load the Extension

For Chrome:

  • Open Chrome and navigate to chrome://extensions/
  • Enable "Developer mode"
  • Click "Load unpacked" and select the Nextension directory

For Firefox:

  • Open Firefox and navigate to about:debugging#/runtime/this-firefox
  • Click "Load Temporary Add-on" and select any file in the Nextension directory
4

Start Using Auto TL;DR

Browse the web as usual. When the extension detects large blocks of text, it will offer to summarize them. Click on the summary indicator to view the summary, or right-click on any page or selection to summarize it manually.
